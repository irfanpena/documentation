Tavus’s Lipsync service, powered by the Hummingbird model, enables you to create natural, synchronized videos by precisely aligning a speaker’s mouth movements with custom audio.

## Key Features
<CardGroup cols={3}>

  <Card title="Natural Lip Synchronization" icon="lips">
    The Replica’s lip movements align precisely with audio, enabling smooth, realistic speech.
  </Card>

  <Card title="Exceptional Identity Preservation" icon="user-check">
    Captures the speaker’s facial features, expressions, and style for a natural, authentic appearance.
  </Card>

  <Card title="Superior Visual Quality" icon="sparkles">
    Every frame is sharp and stable, ensuring professional, polished video output—even at scale.
  </Card>

</CardGroup>


## Getting Started
You can create a personalized lip sync video using the [Tavus Platform](https://platform.tavus.io/) or by following the steps in the [Lipsync API Quickstart Guide](/sections/lipsync/quickstart).
