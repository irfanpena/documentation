## Prerequisites
- Ensure that you have an active Tavus Platform account. If not, sign up [here](https://platform.tavus.io/auth/sign-up).

## Steps

<Steps>
<Step title="Step 1: Create an API key">
1. Go to the [Tavus platform](https://platform.tavus.io/) and click **API Key** in the sidebar.
2. Click **Create New Key**, enter a name and (optionally) specify allowed IP addresses.
3. Click **Create API Key**, then copy and securely save your key.
<Warning>
We cannot recover your API Key if you lose it.
</Warning>
</Step>
